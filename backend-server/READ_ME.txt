for this to be able to work you will need Node.js
Install Node.js
verify installation(run the following in the command prompt)
node -v
npm -v
Change directory to the folder we are working with, example:
cd C:\Users\username\source\repos\PRJ381
create the package.json file to manage dependencies:
npm init -y
Install express web framework:
npm install express
Install the CORS package
npm install cors
run the server before using the website
node server.js
